# 💰 Smart Expense Tracker

### Professional Personal Finance Management System

A production-quality, fully responsive expense tracking dashboard built with **pure HTML5, CSS3, and vanilla JavaScript (ES6)** — no frameworks, no backend. Built as **Task 4** for the **ApexPlanet Software Pvt. Ltd. Web Development Internship**.

---

## 📖 Project Description

Smart Expense Tracker is a premium, SaaS-style personal finance dashboard that helps users manage their income and expenses efficiently. It supports full CRUD transaction management, real-time search/filter/sort, monthly budget tracking, savings monitoring, and rich visual analytics powered by Chart.js — all persisted locally in the browser using LocalStorage.

---

## 🖼️ Project Preview

> Open `index.html` in any modern browser to view the live dashboard — hero overview, dashboard stat cards, transaction form & table, analytics charts, and budget tracker.

---

## ✨ Features

- ✅ **Full CRUD** — Create, Read, Update, and Delete transactions with instant UI updates
- 🔍 **Real-time Search** — Debounced search (300ms) across title, category, and amount
- 🧮 **Advanced Filtering** — Filter by category, transaction type (income/expense), and month
- ↕️ **Flexible Sorting** — Newest, oldest, highest/lowest amount, alphabetical (A–Z / Z–A)
- 📊 **Financial Analytics** — Monthly income/expense, savings, budget usage, average/highest/lowest expense
- 📈 **Interactive Charts (Chart.js)** — Pie chart (expenses by category), bar chart (monthly expenses), line chart (income vs expense trend)
- 💰 **Budget Management** — Set a monthly budget with a live progress bar and over-budget alerts
- 🌗 **Dark / Light Mode** — Theme preference saved and restored automatically
- 📤 **Export** — CSV, JSON, and Print report
- 📥 **Import** — Import transactions from a JSON file with validation and duplicate prevention
- 💾 **Backup & Restore** — Full application data backup/restore from the Settings panel
- 🔔 **Toast Notifications** — Clear success/warning/error feedback for every action
- ♿ **Accessible** — Semantic HTML5, ARIA labels, full keyboard navigation, visible focus states
- 📱 **Fully Responsive** — Optimized for desktop, laptop, tablet, mobile, and small mobile screens
- 🎨 **Premium UI/UX** — Glassmorphism cards, gradient buttons, hover animations, ripple effects, animated counters, scroll progress bar, and back-to-top button

## ⚙️ Installation

No build tools or dependencies are required.

1. Download or clone this repository.
2. Ensure the folder structure remains intact (`css/`, `js/`, `images/` alongside `index.html`).

```bash
git clone https://github.com/karthikyadav/smart-expense-tracker.git
cd smart-expense-tracker
```

---

## ▶️ How to Run

Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari):

- **Option 1:** Double-click `index.html`.
- **Option 2:** Right-click `index.html` → *Open with* → your preferred browser.
- **Option 3 (recommended for development):** Serve it locally using the VS Code **Live Server** extension for hot-reloading.

All data is stored locally in your browser via LocalStorage — no server or internet connection is required after the page assets load.
